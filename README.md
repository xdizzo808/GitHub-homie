## Ritual Flow

    spark -> shell -> forge -> fanout -> birth

| Phase | Description |
|-------|-------------|
| spark | Idea seeds |
| shell | JSON bot/service definitions (blueprints) |
| forge | Logic cores (brains) |
| fanout | Channel adapters (Discord, Telegram, Reddit, API) |
| birth | Complete generated bots/services |

## Node Types

- **Phone Node** - Android/Termux (G1, G2, G3, G5, G6)
- **Hotspot Node** - WiFi hub with captive portal (G2, G3, G4, G5, G6)
- **Cloud Node** - Enterprise broadcast brain (all layers)
