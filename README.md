## GenSpark AI Integration

### Capabilities
- **Sparkpages** - AI-generated custom pages with real-time data
- **Super Agents** - Specialized AI agents for specific tasks
- **Multi-Agent Orchestration** - Coordinate multiple AI agents
- **Real-time Data** - Live information synthesis
- **Custom Agent Creation** - Build domain-specific agents

### Agent Types
- Research Agent - Deep web research and synthesis
- Travel Agent - Trip planning and booking
- Shopping Agent - Product research and comparison
- Coding Agent - Code generation and debugging
- Writing Agent - Content creation and editing
- Data Agent - Data analysis and visualization

### Commands
- genspark:page __tag_topic - Generate a Sparkpage definition
- genspark:agent __tag_type - Generate a custom agent config
- genspark:search __tag_query - Generate search-integrated page

## MetaGPT Integration

### Framework
MetaGPT simulates a software company with role-based agents:
- **ProductManager** - Requirements analysis, PRD generation
- **Architect** - System design, API design, data modeling
- **ProjectManager** - Task breakdown, scheduling, coordination
- **Engineer** - Code implementation, debugging
- **QAEngineer** - Test generation, quality assurance
- **Researcher** - Information gathering, analysis

### Output Artifacts
1. PRD (Product Requirements Document)
2. System Design - Architecture diagrams, API specs
3. Task List - Breakdown of implementation tasks
4. Code Files - Full implementation
5. Tests - Unit and integration tests
6. Documentation - README, API docs

### Commands
- metagpt:project __tag_idea - Generate full project with all roles
- metagpt:role __tag_name - Generate custom role definition
- metagpt:workflow __tag_process - Generate SOP workflow
- metagpt:prd __tag_requirement - Generate PRD document
- metagpt:design __tag_system - Generate system architecture
