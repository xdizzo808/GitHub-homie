## Mission

ARCH ANGEL builds the factory that builds bots. It generates complete, production-ready codebases, templates, blueprints, brains, channels, and documentation - now with multi-agent orchestration and AI page generation.

## Ritual Flow

    spark -> shell -> forge -> fanout -> birth

| Phase | Description |
|-------|-------------|
| spark | Idea seeds |
| shell | JSON bot/service definitions (blueprints) |
| forge | Logic cores (brains) |
| fanout | Channel adapters (Discord, Telegram, Reddit, API) |
| birth | Complete generated bots/services |

## GenSpark AI Integration

### Capabilities
- **Sparkpages** - AI-generated custom pages with real-time data
- **Super Agents** - Specialized AI agents for specific tasks
- **Multi-Agent Orchestration** - Coordinate multiple AI agents

### Commands
- genspark:page __tag_topic - Generate a Sparkpage definition
- genspark:agent __tag_type - Generate a custom agent config
- genspark:search __tag_query - Generate search-integrated page

---

## MetaGPT Integration

### Roles
- **ProductManager** - Requirements analysis, PRD generation
- **Architect** - System design, API design, data modeling
- **Engineer** - Code implementation, debugging
- **QAEngineer** - Test generation, quality assurance

### Commands
- metagpt:project __tag_idea - Generate full project with all roles
- metagpt:prd __tag_requirement - Generate PRD document
- metagpt:design __tag_system - Generate system architecture
